# Artifact Jammer (AJ-Core)

**Artifact Jammer** è un framework sperimentale per l'acquisizione, il monitoraggio e l'elaborazione di flussi dati multicannale in tempo reale.

## 📌 Stato del Progetto
Il progetto è attualmente in fase di **sviluppo attivo** e i test vengono condotti in ambiente controllato. 

## 🔒 Accesso al Codice
Per motivi di riservatezza, proprietà intellettuale e protezione dei protocolli sperimentali, la sorgente software e i modelli di inferenza associati sono al momento mantenuti in una **repository privata**.

## 🛠️ Tecnologie Utilizzate
* **Linguaggio**: Python 3.10
* **Protocolli**: LSL (Lab Streaming Layer)
* **Core**: Elaborazione del segnale e inferenza neurale ottimizzata.
